### Install

**GLFW**

> git clone https://github.com/glfw/glfw && cd glfw && mkdir -p build && cd build && cmake -DCMAKE_C_COMPILER=/usr/bin/gcc -DCMAKE_CXX_COMPILER=/usr/bin/g++ .. && make -j4 && sudo make install

- - - -
